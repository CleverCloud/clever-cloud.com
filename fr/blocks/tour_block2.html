La consommation d'énergie informatique s'effectue de la même manière que la consommation d'électricité. Vous consommez uniquement la ressource informatique utilisée par votre application. part2Title: Sécurisation de vos applications part2Text1: Toutes les applications sont hautement sécurisées :
  • protocole de sécurisation HTTPS par défaut pour toutes les applications
  • sauvegardes de fichiers et de base de données automatiques
  • déploiement via Git ou SFTP
  • surveillance des applications et support 24/7
  • chaque application tourne dans sa machine virtuelle dans une sandbox
  • aucun contact entre les applications
